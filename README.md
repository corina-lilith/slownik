# Slownik Project:

A React website with Polish and Spanish flash cards.

- TypeScript
- React
- Vitest
- MUI
- NodeJS
- PostgreSQL

## Installation

- Clone the repo onto your device
- Run `npm install`

### Testing

- Run `npm run test`

## Development

- Run `npm run dev`

## Release

- Run `npm run build`
- The compiled output will be available in the `dist` folder
